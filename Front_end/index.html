About Intrusion Detection System

System information and technical details

System Overview

This is a professional, machine learning-based Intrusion Detection System designed to monitor network traffic, detect anomalies, and identify various types of cyber attacks in real-time. The system uses advanced algorithms to analyze network packets and provide actionable security insights.

Technologies Used

  • Backend: Python, Flask, Scikit-learn
  • Frontend: HTML5, CSS3, JavaScript (ES6+)
  • Visualization: Chart.js
  • ML Algorithm: Random Forest Classifier
  • Dataset: NSL-KDD & CICIDS2017

Dataset Information

The system is trained on industry-standard datasets including NSL-KDD and CICIDS2017, which contain diverse network traffic patterns and various attack scenarios such as DoS, DDoS, Port Scanning, Brute Force, and more.

Model Performance

Accuracy 99.5%
Precision 99.3%
Recall 99.6%
F1 Score 99.4%

Detected Threats

  • DoS/DDoS: Denial of Service attacks
  • Probe: Port scanning and reconnaissance
  • R2L: Remote to Local attacks
  • U2R: User to Root privilege escalation
  • Brute Force: Password cracking attempts
